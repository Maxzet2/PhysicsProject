# PhysicsProject

Разделённый проект: Core (домен), Application (use-cases), Infrastructure (БД, внешние сервисы), Api (ASP.NET Core Web API).

## Запуск локально

```bash
cd PhysicsProject
dotnet build
cd PhysicsProject.Api
dotnet run
```

API будет доступно на http://localhost:5120, OpenAPI при Development.
